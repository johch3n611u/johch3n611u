# 共筆操作方式

## clone 專案至本地

ctrl + p

&gt;git:clone

github Clone with URL

## 開新檔

1. ctrl+e 查想寫的內容的關鍵字，是否已經有共筆。
2. 如果有就先查看一下內容，看大綱是否沒有想共筆的內容，如果沒有就共筆，有的話就去看有沒有少筆記。

## 建議編寫格式為 ( 盡量或看到沒有就幫忙補 )

1. 內容: 大綱 -> 內文
2. 檔名: 文章討論主題_起始日期.md

## 更新推檔

1. 每次 push 前先 pull 修改衝突後再 push。
2. 修改衝突的 commit 最好別打亂碼，方便辨識修改了什麼。
