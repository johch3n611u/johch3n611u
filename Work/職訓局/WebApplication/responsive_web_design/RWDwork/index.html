台灣指標民調

2020大選在即

柯文哲拋大巨蛋以6萬人上線後再改 林洲民提出質疑

〔記者楊心慧／台北報導〕台北市長柯文哲昨獲邀到工商協進會理監事會議進行專題演講，被企業代表質疑卡蛋這麼久問題在哪？未料，柯拋出大巨蛋可用環評人數6萬人「上線以後可以再改」，與喬蛋錄音檔曝光前，市府內部積極將容留人數提高到7萬人的態度截然不同，讓林洲民質疑：「這不就是去年9月20日我主持的都審會決議嗎？那為什麼柯的前辦公室主任、現任北捷董事長李文宗還要長期干擾都發局、都審會替遠雄爭取到7萬以上？」

報道：美朝河內峰會後，北韓處決首席談判代表等五名高官

韓國《北韓日報》(Chosun Ilbo)周四報道稱，今年2月份金正恩(Kim Jong Un)與美國總統特朗普(Donald Trump)在越南舉行的峰會無果而終，之後北韓處決了包括首席談判代表在內的五名高官。該報報道，另有一名高官被送進勞改營。 《北韓日報》稱，河內峰會北韓談判團隊的領隊金赫哲(Kim Hyok Chol)與另外四名高級官員今年3月份被槍決，此前他們被控為美國從事間諜活動。

蔡英文打完賴就會來打柯？ 柯文哲：我聽了好害怕！

柯文哲今天早上主持里長座談會前再談，為什麼覺得蔡英文跟賴清德打完會來打你？柯文哲說，這也是人家跟他講的，「人家說今天小英怎麼打賴清德以後就怎麼打你，我聽了好害怕！」 媒體又問，怎麼看與賴清德民調之中，年輕族群支持度贏了40％？柯文哲不以為然說，民調看看就好，且怎麼不說我那個長者的族群輸他40個百分點？

答應同框恰吉了！ 王世堅：樂意當恰吉恫嚇市長、官員

〔記者郭安家／台北報導〕台北市議員王世堅說，即將上映的《恰吉》是他人生第一次的做商業宣傳；約莫30年前曾和妻子看鬼娃第一集，多年來不看恐怖片，沒用網路、智慧型手機，渾然不知過去造型及齜牙咧嘴模樣被製作成「恰吉」kuso圖，還有那些網路討論。王世堅詮釋說，「為市民把關、捍衛他們權益，我樂意當恰吉，恫嚇市長、官員，讓有權有勢有資源者不要作威作福」。

假日選總統？韓國瑜昨密會5人小組 網諷：週三小週末啦！

〔即時新聞／綜合報導〕上任僅半年的高雄市長韓國瑜有意帶職參選總統，還稱「難道我不能一面專心在市政，用假日去選嗎？當然可以」。不過，韓昨在左營蓮潭會館會晤國民黨五人小組，被問到參選意願，他重申「「Yes, I do.」。臉書粉專「打馬悍將」貼文質疑，「禮拜三，上班日」，不少網友也留言響應。

蔡英文：我的「辣」在於對很多事很堅持 也很能忍

蔡英文總統今年初強勢回應大陸一國兩制台灣方案發言後，有了「辣台妹」稱號，這次民進黨總統初選，蔡總統也以辣台妹自居。蔡總統今天接受廣播節目專訪時說，她的辣，在於她對很多事很堅持很堅定，再大的困難都會去面對。 蔡總統表示，她也很能夠忍，忍到撥雲見日。這幾年來，她在執政過程中也歷經一些困難，但經過三年，困難一樣一樣解決，人民也都感受到執政成績。過去有很多人對她不諒解的地方，「突然覺得過去三年的努力與忍隱是值得的」。

觀察站／國民黨網內互打 最壞兩結局

國民黨總統初選開打，朱立倫、郭台銘昨天將矛頭指向黨中央，加上王金平也喊話團結，雖然參選人出發點及盤算各有不同，但從初選機制到黨內選邊站，反誰、黑誰的攻擊已近白熱化，主辦選務的黨中央成為眾矢之的。 總統初選制度一改再改，對朱立倫和王金平而言，的確極不公平，不滿可以理解，黨中央也無可辯駁。但對郭台銘和韓國瑜而言，雖然背後陰謀論同樣不少，但至少兩人的出場，某種程度是在公平的起跑線，且不管比人勢、財勢，郭、韓陣營恐怕自認更在「黨之上」，但也因如此，與黨中央的緊張關係，恐更不下朱、王。

基層憂 朱立倫：國民黨2020總戰略在哪

國民黨總統初選競爭激烈，黨內謠言滿天飛，引發參選人不滿。國民黨中央提名五人小組昨拜會新北市前市長朱立倫，朱當面提醒，他近來走訪基層，很多人都說看不明白黨中央「2020選舉的總戰略到底是什麼」。
